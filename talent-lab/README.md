# Talent Lab

> Универсальная система: из сырого человека (характер + история + интересы) получить 3–5 реалистичных направлений монетизации, выбрать одно, упаковать его под рынок и провалидировать на реальных людях.
>
> Работает для любых профессий — от Frontend Lead до лашмейкера и воспитательницы.
> Конечная цель (5 лет): на основе пилота вырастить собственную школу/программу.

## Как использовать на одном человеке (типовой проход)

1. Договориться о времени, отправить `01-intake.md` и попросить заполнить.
2. Совместно или асинхронно собрать данные.
3. Вставить ответы intake в начало любого AI-чата вместе с `02-discovery-prompt.md`. Получить карту направлений.
4. Обсудить карту с человеком, выбрать топ-1 вектор.
5. В зависимости от типа вектора — взять `03-packager-corporate.md` или `04-packager-personal-brand.md`. Прогнать выбранный вектор через AI.
6. По упаковке — 2-недельные bridge-эксперименты (см. раздел в Discovery-промпте).

## Как использовать на группе (пилот → школа)

1. Отобрать 3–4 человека по критериям из `05-pilot-program.md`.
2. Запустить 6-недельный цикл с замером метрик до/после.
3. На 10-й неделе собрать кейсы и видео-отзывы.
4. Решить: зелёный свет (платный поток) / жёлтый (1-на-1 консалтинг) / пивот / стоп.
